# CACHE FINDER
---

> Backend-сервис, который позволяет искать **горячие данные** в кэше, хранящимся в Redis

Сервис показывает базовую архитектуру под **Service-layer**, соответствующий стандартам 
**FastAPI**, а также испоользует принципы *ООП* и *SOLID*.

## Быстрый старт

```bash
 docker compose up -d
```

| Адрес                                                    | Назначение |
|----------------------------------------------------------|------------|
